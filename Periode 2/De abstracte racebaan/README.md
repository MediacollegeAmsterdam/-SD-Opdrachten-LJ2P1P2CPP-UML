# De abstracte racebaan

Dit project bestaat uit 4 classes en hoeft geen echte data te bevatten.
-	RaceTrack.h en RaceTrack.cpp
-	Race.h en Race.cpp
-	Car.h en Car.cpp
-	Driver.h en Driver.cpp

**Alle variabelen zijn private**
